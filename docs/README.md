# Documentation

Material is grouped by topic. Paths in code comments and runbooks use repo-root form, e.g. `docs/weekly-contests/weekly-contests-phase4-adr.md`.

| Folder | Contents |
|--------|----------|
| **`weekly-contests/`** | Weekly contests ADRs, Jira trackers, Firestore schemas, API notes, ops hooks, runbooks, UI walkthrough |
| **`leaderboards/`** | Leaderboards ADR, schema, indexes, rate limits, runbooks, cohort QA; [contest live leaderboard roadmap](leaderboards/leaderboards-contest-live-roadmap.md) |
| **`admin/`** | Admin dashboard security, operator ops (claims), Jira backlog |
| **`platform/`** | Firestore rules deploy / mirror, environment matrix, gameplay stats & observability, stats reconciliation |
| **`payments/`** | Stripe keys, Checkout, server integration |
| **`product/`** | Product roadmap (contests & payments) |
| **`fixtures/`** | Example JSON for staging seeds and verification scripts |
